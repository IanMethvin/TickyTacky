Ticky Tacky

Game Description

Welcome to Ticky Tacky, an HTML 5 based implementation of the original Tic Tac Toe game.

Built with:

  • HTML5 Canvas
  • jQuery
  • CSS

Brief description of the game:

  • No time limit
  • Board waits for user input before responding
  • Responding AI uses random move algorithm
  • Game tracks win status for either user or AI
  • Game ends when one symbol (either X or O) achieves the normal tic-tac-toe win requirements.
